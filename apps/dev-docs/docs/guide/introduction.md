# Introduction

Welcome to the Manyumba developer documentation!

## What is Manyumba?

Manyumba is a modern web application platform built with:

- **Nuxt** - The intuitive Vue framework
- **TypeScript** - Type-safe development
- **Turborepo** - High-performance build system for monorepos
- **PostgreSQL** - Robust database layer
- **Docker** - Containerized local development

## Project Structure

```
manyumba/
├── apps/
│   ├── web/          # Main Nuxt application
│   ├── user-docs/    # User-facing documentation
│   ├── dev-docs/     # Developer documentation (this site)
│   └── base/         # Base configuration
├── packages/
│   ├── types/        # Shared TypeScript types
│   └── utils/        # Shared utilities
└── turbo.json        # Turborepo configuration
```
